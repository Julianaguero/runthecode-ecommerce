Desarrollo de un proyecto de e-commerce en un mono-repositorio multipaquete, abarcando tanto backend como frontend.

Backend:
- Stack: Node.js, Express, Mongoose, MongoDB.
- Diseño y desarrollo de una API Rest con dos microservicios: Productos y Carrito de compras, cada uno con operaciones CRUD completas.
- Atomización de archivos para mejorar la visualización.
- Manejo de errores en las rutas para garantizar el correcto funcionamiento del servicio y mejorar la comunicación con el frontend.
- En desarrollo: Microservicio de gestión de pedidos y gestión de usuarios con JWT.

Frontend:
- Stack: React, Tailwind CSS, TypeScript.
- Interfaz intuitiva y responsive con 3 puntos de corte. En desarrollo: Hamburger menu.
- Página principal optimizada utilizando framer-motion para interactividad.
- Barra de filtros en el Shop y barra Breadcrums para mejorar la navegación del usuario. En desarrollo: Mejorar y optimizar la barra de filtros e incorporar una barra de búsqueda de productos.
- Carrito de compras intuitivo que permite al usuario ver su carrito desde la barra de navegación y desde una página anterior a la pasarela de pagos. En desarrollo: Optimizar la apertura y cierre del pop-up del carrito y desarrollar íntegramente la plataforma de pagos.
- En desarrollo: Ruta collections, donde se mostrarán productos relacionados con la marca del producto clickeado en la Home page. Interfaz de login y opciones de usuario.
